Cómo compartirlo bien

Usá siempre el link principal para que la experiencia sea prolija.

  • Compartí la portada principal y no los tres links sueltos, salvo que necesites enviar uno específico.
  • Usá este sitio como base para WhatsApp, Instagram, bio, Link-in-bio o presentaciones privadas.
  • Cuando actualices tus HTML en GitHub, el sitio se actualiza solo.
  • Más adelante podés sumar formulario, botón de registro, contacto o branding visual más fuerte.
